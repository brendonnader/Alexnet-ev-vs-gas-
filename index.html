Project Overview

This project uses a convolutional neural network (AlexNet) to classify vehicles as electric or gas-powered based on images. The goal is to show how deep learning can support real-world transportation and sustainability analytics.

What You’ll Find on This Site

  • Problem & Motivation: Why detecting EV vs gas vehicles matters.
  • Technical Approach: How AlexNet works, how the data is preprocessed, and how the model is trained.
  • Results & Visualizations: Training curves, confusion matrix, example predictions, and key takeaways.

Tools Used

  • Python + PyTorch (Google Colab)
  • AlexNet (pretrained on ImageNet, fine-tuned on EV vs Gas dataset)
